The repository contains code files for accessing peripherals and interfacing sensors (inertial measurement unit, encoder, lcd , 
relay drivers) on Atmel Atmega128.
The programmes are written in BASCOM IDE , for avr platform.
The programmes are tested and verified on custom made board for Atmega128.
The programmes can be converted for Atmega8/16/32 by changing the header line in code to specific IC name .
